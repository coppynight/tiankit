# tiankit

Kai 的研发工具包（toolkit），包含两个核心工具：

- **tianji**：把想法转成高质量、可落地的项目计划（规格/里程碑/验收/风险）。
- **tiangong**：把项目计划拆解并自动化执行（PM/Worker/Reviewer/Watchdog 协作，证据链驱动）。

> 目标：让项目协作更可控 —— 计划可验证、执行可追踪、结果可验收。
